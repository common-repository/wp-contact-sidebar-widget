== Description ==

WP-Contact puts a contact form in your sidebar. you can read and reply to message in the admin area under the manage tab!

== Installation ==

1. Upload `wp-contact/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. go to your widgets admin tools located in the admin section under the presentation tab and drag the contact widget into the list.If you are using a theme that doesn’t support widgets just add this line of code to your sidebar.php page <?php widget_wp_contact(); ?>
